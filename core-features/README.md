# Core Features

> Taolend is a P2P (peer-to-peer) lending platform that connects lenders and borrowers through an Offer-based matching mechanism.

## Feature Overview
- **[Borrow](borrow.md)**: How to borrow TAO using ALPHA as collateral
- **[Lend](lend.md)**: How to create Offers and earn interest

---

## Workflow

### Lender Workflow

1. **Bridge TAO** →
2. **Register Account** →
3. **Deposit TAO** →
4. **Create Loan Offer** →
5. **Wait for Borrowers** →
6. **Earn Interest** →
7. **Withdraw**

### Borrower Workflow

1. **Bridge ALPHA** →
2. **Register Account** →
3. **Deposit ALPHA** →
4. **Borrow Assets** →
5. **Receive TAO** →
6. **Repay Loan or Refinance**
